Lecture des Chandeliers

Apprenez à lire les chandeliers japonais.

15 min Débutant

📜 Histoire: Des marchands de riz japonais aux traders modernes

Les chandeliers japonais ont été inventés il y a plus de 300 ans par Munehisa Homma, un légendaire marchand de riz japonais. À une époque sans ordinateurs ni internet, il a développé un système visuel pour analyser les prix qui reste le standard mondial aujourd'hui.

Homma a fait fortune en comprenant que le prix reflète les ÉMOTIONS des participants: peur, avidité, espoir, panique. Chaque chandelier raconte une histoire de la bataille entre acheteurs et vendeurs.

Aujourd'hui, 95% des traders professionnels utilisent les chandeliers comme base de leur analyse technique. Maîtriser leur lecture est FONDAMENTAL.

│         │         │ ← Corps          │         │ ← Corps                 │

│         └────┬────┘                  └────┬────┘                         │
│              │ ← Mèche basse              │ ← Mèche basse                │
│              │   (Open - Low)             │   (Close - Low)              │
│              │                            │                              │
│                                                                          │
├─────────────────────────────────────────────────────────────────────────┤
│ LES 4 PRIX:                                                              │
│ • OPEN (Ouverture): Premier prix de la période                          │
│ • HIGH (Plus haut): Prix maximum atteint                                │
│ • LOW (Plus bas): Prix minimum atteint                                  │
│ • CLOSE (Clôture): Dernier prix de la période                           │
├─────────────────────────────────────────────────────────────────────────┤
│ INTERPRÉTATION DES MÈCHES:                                               │
│ • Longue mèche haute = Vendeurs ont repoussé le prix (rejet)            │
│ • Longue mèche basse = Acheteurs ont défendu le prix (support)          │
│ • Pas de mèche = Conviction forte dans la direction                      │

🎯 Le CONTEXTE est ROI - Même pattern, significations différentes

Un pattern de chandelier SEUL ne signifie RIEN. C'est le CONTEXTE qui lui donne sa puissance. Voici les règles d'or:

┌─────────────────────────────────────────────────────────────────────────┐
│                    RÈGLES D'INTERPRÉTATION                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│  RÈGLE 1: POSITION DANS LA TENDANCE                                      │
│  • Hammer en bas de range = FORT signal d'achat                          │
│  • Hammer au milieu = Peu significatif                                   │
│  • Hammer en haut = IGNOREZ-LE                                           │
│                                                                          │
│  RÈGLE 2: NIVEAU TECHNIQUE                                               │
│  • Pattern sur support/résistance = Puissance x3                         │
│  • Pattern sur niveau GEX = Puissance x5                                 │
│  • Pattern "dans le vide" = Fiabilité réduite                            │
│                                                                          │
│  RÈGLE 3: VOLUME                                                         │
│  • Pattern + Volume élevé = Confirmation forte                           │
│  • Pattern + Volume faible = Méfiez-vous                                 │
│                                                                          │
│  RÈGLE 4: TIMEFRAME                                                      │
│  • Pattern sur Daily = Plus fiable que sur 1min                          │
│  • Toujours vérifier le timeframe supérieur                              │
│                                                                          │
│  RÈGLE 5: CONFIRMATION                                                   │
│  • N'entrez JAMAIS sur le pattern seul                                   │
│  • Attendez la bougie suivante pour confirmer                            │
│  • Ex: Hammer → Attendez bougie verte pour confirmer                     │
│                                                                          │
└─────────────────────────────────────────────────────────────────────────┘

❌ ERREURS COMMUNES avec les chandeliers

  • •❌ ERREUR #1: Trader un pattern sans contexte → Un hammer dans le vide ne vaut rien
  • •❌ ERREUR #2: Ignorer la confirmation → Entrer avant la bougie de confirmation = 50% de faux signaux
  • •❌ ERREUR #3: Utiliser des timeframes trop petits → Les patterns 1min sont du bruit
  • •❌ ERREUR #4: Oublier le volume → Un pattern sans volume n'a pas de conviction
  • •❌ ERREUR #5: Chercher des patterns partout → Le marché est souvent "neutre", pas de pattern
  • •❌ ERREUR #6: Confondre les patterns → Hammer vs Hanging Man = même forme, contexte différent
  • •❌ ERREUR #7: Sur-compliquer → Maîtrisez 5 patterns à fond plutôt que 20 superficiellement

✅ POINTS CLÉS à retenir

  • •✅ Les chandeliers racontent l'histoire de la bataille acheteurs vs vendeurs
  • •✅ Les 4 prix (OHLC) sont la base: Open, High, Low, Close
  • •✅ Les mèches indiquent les rejets de prix (offre/demande)
  • •✅ Patterns de retournement: Hammer, Shooting Star, Doji, Engulfing
  • •✅ Patterns de continuation: 3 White Soldiers, 3 Black Crows, Marubozu
  • •✅ Le CONTEXTE est crucial: niveau, volume, timeframe, tendance
  • •✅ Toujours attendre la CONFIRMATION avant d'entrer
  • •✅ Sur Futures, le graphique 5min est optimal pour les patterns
