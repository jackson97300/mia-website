OrderFlow Basics

Introduction à l'OrderFlow.

20 min Avancé

🎯 OrderFlow: Voir l\

L'OrderFlow vous permet de voir ce qui se passe RÉELLEMENT sur le marché: qui achète, qui vend, à quelle agressivité.

Au lieu de deviner la direction à partir des bougies, vous voyez les ordres RÉELS qui bougent le prix.

C'est l'outil des traders professionnels pour comprendre l'intention derrière chaque mouvement.

📊 SCHÉMA: Composantes de l\

┌─────────────────────────────────────────────────────────────────────────┐
│              COMPOSANTES DE L'ORDERFLOW                                  │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│  1. DELTA (Volume Directionnel)                                         │
│  ════════════════════════════════                                        │
│  Delta = Achats agressifs - Ventes agressives                           │
│                                                                          │
│  • Delta POSITIF = Plus d'achats que de ventes                          │
│  • Delta NÉGATIF = Plus de ventes que d'achats                          │
│  • Delta doit CONFIRMER le mouvement de prix                            │
│                                                                          │
│  Prix monte + Delta positif = Mouvement SAIN ✅                         │
│  Prix monte + Delta négatif = DIVERGENCE ⚠️ (potentiel reversal)        │
│                                                                          │
│  2. CVD (Cumulative Volume Delta)                                        │
│  ════════════════════════════════                                        │
│  CVD = Somme du delta sur la période                                    │
│  • Montre la pression acheteuse/vendeuse cumulée                        │
│  • Divergence CVD/Prix = Signal de retournement                         │
│                                                                          │
│  3. DOM (Depth of Market)                                                │
│  ════════════════════════════════                                        │
│  • Montre les ordres en attente (bids/asks)                             │
│  • Gros bids = Support potentiel                                        │
│  • Gros asks = Résistance potentielle                                   │
│  • ATTENTION: Peuvent être retirés (spoofing)                           │
│                                                                          │
└─────────────────────────────────────────────────────────────────────────┘

⚡ Divergences OrderFlow: Signaux de retournement

┌─────────────────────────────────────────────────────────────────────────┐
│              DIVERGENCES ORDERFLOW                                       │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│  DIVERGENCE BAISSIÈRE:                                                   │
│  ═══════════════════════                                                 │
│  Prix:  ────────/‾‾‾\                                                   │
│  Delta: ────────\___/     Delta baisse alors que prix monte             │
│                                                                          │
│  Signification: Les acheteurs s'essoufflent                             │
│  Action: Préparer des SHORTS                                            │
│                                                                          │
│  DIVERGENCE HAUSSIÈRE:                                                   │
│  ═══════════════════════                                                 │
│  Prix:  ────────\___/                                                   │
│  Delta: ────────/‾‾‾\     Delta monte alors que prix baisse             │
│                                                                          │
│  Signification: Accumulation discrète                                   │
│  Action: Préparer des LONGS                                             │
│                                                                          │
│  ⚠️ IMPORTANT: La divergence seule n'est pas un signal d'entrée.       │
│  Elle vous PRÉPARE. Attendez la confirmation (pattern, niveau).         │
│                                                                          │
└─────────────────────────────────────────────────────────────────────────┘

❌ ERREURS et ✅ POINTS CLÉS

  • •❌ Trader uniquement sur le delta → C'est un CONFIRMATEUR, pas un signal
  • •❌ Ignorer le DOM → Il montre où sont les gros ordres
  • •❌ Suivre aveuglément les gros ordres → Spoofing existe
  • •✅ Delta = Achats - Ventes agressives = Direction de la pression
  • •✅ CVD = Delta cumulé = Vue d'ensemble de la journée
  • •✅ Divergence OrderFlow = Signe d'essoufflement
  • •✅ OrderFlow CONFIRME, ne prédit pas

📝 À RETENIR

  • •🎯 OrderFlow = Voir l'ADN du marché (qui achète/vend réellement)
  • •📊 Delta = Achats agressifs - Ventes agressives
  • •📊 CVD = Delta cumulé → Vue d'ensemble de la journée
  • •⚠️ Divergence Delta/Prix = Signal d'essoufflement (préparez-vous)
  • •🔍 Absorption = Volume élevé + Delta opposé = Piège!
  • •⚡ Initiative > Responsive: Celui qui frappe le marché contrôle
  • •⛔ OrderFlow CONFIRME, ne PRÉDIT jamais - Attendez le niveau
