Call Resistance & Put Support

L'impact des options.

12 min Intermédiaire

🎯 Les OPTIONS créent des niveaux INVISIBLES

Voici un secret: les options sur indices créent des niveaux de support/résistance INVISIBLES sur les graphiques traditionnels.

Quand des milliards de $ d'options sont concentrés à un strike, les market makers qui les ont vendues doivent HEDGER. Ce hedging crée une force magnétique.

Comprendre Call/Put Walls vous donne un avantage énorme: vous voyez des niveaux que 90% des traders ignorent.

📊 SCHÉMA: Call Wall = Résistance

┌─────────────────────────────────────────────────────────────────────────┐
│              CALL WALL - RÉSISTANCE INVISIBLE                            │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│  Prix ES                                                                 │
│    │          ════════════════  ← CALL WALL (ex: 5100)                  │
│    │         ↗↗ RÉSISTANCE ↗↗                                          │
│    │    ╱‾╲                                                             │
│    │   ╱   ╲   Le prix est REPOUSSÉ                                     │
│    │  ╱     ╲                                                           │
│    │ ╱       ╲                                                          │
│    │╱         ╲                                                         │
│                                                                          │
│  POURQUOI?                                                               │
│  1. MM ont VENDU des calls au strike 5100                               │
│  2. Prix monte vers 5100 → MM perdent                                   │
│  3. Pour hedger, MM VENDENT des futures                                 │
│  4. Cette pression vendeuse = RÉSISTANCE                                │
│                                                                          │
│  → Le Call Wall agit comme un PLAFOND                                   │
└─────────────────────────────────────────────────────────────────────────┘

│                                                                          │

│    │    ╲_╱                                                             │
│    │         ↙↙ SUPPORT ↙↙                                             │

│  4. Cette pression acheteuse = SUPPORT                                  │
│                                                                          │
│  → Le Put Wall agit comme un PLANCHER                                   │

💰 Stratégies sur Call/Put Walls

┌─────────────────────────────────────────────────────────────────────────┐
│              STRATÉGIES DE TRADING                                       │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│  STRATÉGIE 1: REBOND SUR PUT WALL                                       │
│  • Prix approche Put Wall + pattern de retournement                     │
│  • Entrée: Au-dessus du high du pattern                                 │
│  • Stop: Sous le Put Wall (marge 3-5 pts)                               │
│  • Target: Prochain Call Wall ou GEX                                    │
│                                                                          │
│  STRATÉGIE 2: REJET SUR CALL WALL                                       │
│  • Prix approche Call Wall + pattern baissier                           │
│  • Entrée: Sous le low du pattern                                       │
│  • Stop: Au-dessus du Call Wall (marge 3-5 pts)                         │
│  • Target: Prochain Put Wall ou GEX                                     │
│                                                                          │
│  ⚠️ IMPORTANT:                                                          │
│  • Les walls CHANGENT chaque jour                                       │
│  • Besoin de données fraîches (MenthorQ)                                │
│  • Toujours attendre CONFIRMATION avant entrée                          │
│                                                                          │
└─────────────────────────────────────────────────────────────────────────┘

❌ ERREURS et ✅ POINTS CLÉS

  • •❌ Traiter tous les walls également → Certains ont 10x plus d'OI
  • •❌ Ignorer l'expiration → Un wall 0DTE disparaît à 22h
  • •❌ Entrer sans confirmation → Le prix peut transpercer avant de rebondir
  • •✅ Call Wall = Résistance créée par hedging des MM
  • •✅ Put Wall = Support créé par hedging des MM
  • •✅ Plus l'Open Interest est élevé, plus le wall est fort
  • •✅ La stratégie principale est le REBOND, pas le breakout
