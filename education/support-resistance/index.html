Support & Résistance

Identifiez les niveaux clés.

12 min Débutant

🎯 Introduction: Les fondations du Price Action

Les supports et résistances sont les concepts les PLUS FONDAMENTAUX de l'analyse technique. Sans les comprendre, vous tradez à l'aveugle.

Un support est un niveau de prix où la DEMANDE est suffisamment forte pour arrêter/inverser une baisse. Une résistance est un niveau où l'OFFRE est suffisamment forte pour arrêter/inverser une hausse.

Ces niveaux existent parce que les traders ont une MÉMOIRE. Ils se souviennent des prix importants et agissent en conséquence quand le marché y revient.

🔍 Comment IDENTIFIER les bons niveaux

┌─────────────────────────────────────────────────────────────────────────┐
│              CHECKLIST D'IDENTIFICATION                                  │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│  ☑️ ÉTAPE 1: Identifier les zones de RÉACTION                            │
│  • Où le prix a-t-il REBONDI dans le passé?                             │
│  • Minimum 2 touches (idéalement 3+)                                     │
│  • Réactions CLAIRES (pas juste une pause)                               │
│                                                                          │
│  ☑️ ÉTAPE 2: Vérifier le VOLUME                                          │
│  • Y a-t-il eu une augmentation de volume à ce niveau?                   │
│  • Volume = conviction des participants                                  │
│                                                                          │
│  ☑️ ÉTAPE 3: Confirmer sur PLUSIEURS timeframes                          │
│  • Un niveau visible sur Daily ET 4H = très fort                         │
│  • Un niveau visible seulement sur 5min = faible                         │
│                                                                          │
│  ☑️ ÉTAPE 4: Croiser avec niveaux INSTITUTIONNELS                        │
│  • Support technique + GEX level = CONFLUENCE = entrée idéale           │
│  • Plus il y a de confluence, plus le niveau est fort                   │
│                                                                          │
│  ☑️ ÉTAPE 5: Considérer le CONTEXTE                                      │
│  • Support dans une tendance haussière = plus susceptible de tenir      │
│  • Support dans une tendance baissière = plus susceptible de casser     │
│                                                                          │

💰 Stratégies de Trading sur les niveaux

┌─────────────────────────────────────────────────────────────────────────┐
│                    2 STRATÉGIES PRINCIPALES                              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│  📈 STRATÉGIE 1: REBOND (Mean Reversion)                                 │
│  ──────────────────────────────────────                                  │
│  Principe: Acheter au support, vendre à la résistance                   │
│                                                                          │
│  Conditions pour un LONG sur support:                                    │
│  1. Prix approche un support identifié                                  │
│  2. Pattern de retournement (hammer, engulfing...)                      │
│  3. Volume qui augmente au toucher                                       │
│  4. RSI en survente (<30) = bonus                                        │
│                                                                          │
│  Entrée: Au-dessus du high du pattern de retournement                   │
│  Stop: Sous le support (avec marge de 2-3 ticks)                        │
│  Target: Prochaine résistance ou niveau GEX                              │
│                                                                          │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│  📉 STRATÉGIE 2: BREAKOUT (Trend Following)                              │
│  ──────────────────────────────────────                                  │
│  Principe: Suivre la cassure d'un niveau majeur                         │
│                                                                          │
│  Conditions pour un LONG sur breakout résistance:                       │
│  1. Prix casse AU-DESSUS d'une résistance claire                        │
│  2. VOLUME en forte augmentation (+50% min)                              │
│  3. Bougie de cassure avec CORPS (pas juste mèche)                       │
│  4. Pas de news majeure dans les 30 prochaines minutes                  │
│                                                                          │
│  Entrée: Au retest de la résistance devenue support                     │
│  Stop: Sous le nouveau support (ancienne résistance)                    │
│  Target: Extension du mouvement (1.5x à 2x la distance)                 │
│                                                                          │
└─────────────────────────────────────────────────────────────────────────┘

❌ ERREURS COMMUNES avec S/R

  • •❌ ERREUR #1: Tracer des niveaux PARTOUT → Le graphique devient illisible. Max 5-7 niveaux pertinents.
  • •❌ ERREUR #2: Ignorer le contexte de tendance → Un support en tendance baissière est plus susceptible de casser.
  • •❌ ERREUR #3: Entrer AVANT confirmation → Le prix peut traverser un niveau puis revenir. Attendez le pattern.
  • •❌ ERREUR #4: Stop trop serré → Mettez le stop SOUS/AU-DESSUS du niveau, pas pile dessus.
  • •❌ ERREUR #5: Ne pas actualiser les niveaux → Un niveau cassé devient l'inverse. Mettez à jour!
  • •❌ ERREUR #6: Traiter tous les niveaux également → Un GEX level >> un swing high aléatoire.
  • •❌ ERREUR #7: Oublier le volume → Sans volume, un "rebond" n'est qu'une pause.

✅ POINTS CLÉS à retenir

  • •✅ Support = zone d'achat (demande), Résistance = zone de vente (offre)
  • •✅ Plus un niveau est testé, plus il est significatif
  • •✅ Un niveau cassé S'INVERSE: support → résistance et vice-versa
  • •✅ La CONFLUENCE de niveaux = entrées les plus fiables
  • •✅ Toujours confirmer avec pattern + volume avant d'entrer
  • •✅ Hiérarchie: Niveaux GEX > PDH/PDL > Swing Highs/Lows
  • •✅ Le retest après breakout est souvent meilleur que le breakout
  • •✅ Ne tradez pas "dans le vide" - uniquement près de niveaux clés
